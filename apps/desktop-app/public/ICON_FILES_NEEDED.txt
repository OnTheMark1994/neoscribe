REQUIRED ICON FILES FOR ELECTRON BUILDS
=========================================

Current Status:
✅ icon.ico - Windows icon (EXISTS)
❌ icon.icns - macOS icon (MISSING - will cause macOS build to fail)
❌ icon.png - Linux icon (MISSING - may cause Linux build to fail or use default)

To Fix:
-------

Option 1 - Generate from existing icon.ico:
1. Use online converter: https://cloudconvert.com/ico-to-icns
2. Convert icon.ico to icon.icns (macOS)
3. Convert icon.ico to PNG (1024x1024) and save as icon.png (Linux)

Option 2 - Create proper icons:
1. Create a 1024x1024 PNG master icon
2. Use electron-icon-builder or similar tool to generate all formats

Option 3 - Quick fix for testing:
1. Copy logo512.png as icon.png (temporary)
2. Skip macOS builds until icon.icns is ready

Files should be placed in:
apps/desktop-app/public/

Without these files, GitHub Actions will fail when building for macOS and possibly Linux.
